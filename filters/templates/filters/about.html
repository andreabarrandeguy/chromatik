About the project

Hi! I'm Andrea

I developed the "Chromatik" tool for my Harvard's CS50 final project.

During the course I learned so many new things, but one that stuck in my mind was the posibility to alter the RGB values of each individual pixel within an image.

Specially because it reminded me about cross processing film... and how expensive it has become... and how it would be amazing to have a tool to "test" if a scene would benefit from certain cross processes.

I am really satisfied with the filters developed so far ("Purple" and "Turquoise"), but I would want to keep adding more to the selection.
